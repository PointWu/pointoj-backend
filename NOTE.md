

# 系统功能梳理


- 题目模块
  - 创建题目
  - 删除题目
  - 修改题目
  - 搜索题目
  - 在线做题


- 判题信息枚举
  - Accepted 成功
  - Wrong Answer 答案错误
  - Compile Error 编译错误
  - Memory Limit Exceeded 超时
  - Presentation Error 展示错误
  - Output Limit Exceeded 输出溢出
  - Waiting 等待中
  - Dangerous Operation 危险操作
  - Runtime Error 运行错误（用户程序问题）
  - System Error 系统错误 （做系统人的问题）
